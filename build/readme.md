# vue-client-pagination

Easy Cds integration for Vue.

## Install

```sh
$ yarn add @gigigo/cds-lib
```

## Usage

```javascript
// main.js
import CdsLib from "@gigigo/cds-lib";
Vue.use(CdsLib);
```

```html
<!-- Vue template -->
<CdsLib>
  <div slot-name="loading">
    Loading...
  </div>
</CdsLib>
```
